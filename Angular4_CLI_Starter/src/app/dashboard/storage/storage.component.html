WebTorrent

WebTorrent does not support all torrent protocols. If you seed a file from this app it will be available for anyone else using WebTorrent as long as you are connected to the network or someone else has a copy downloaded.

You can alternatively upload your files to IPFS and share the torrent file. This allows people to seed and share the content even while you are offline. Any torrent files created with content on IPFS cannot be censored.
